## 🔍 OCR 결과 샘플

### 정확한 추출 사례
```
원본:   "4. 영어와 우리말 표현이 일치하지 않는 것은?"
OCR:    "4. 영어와 우리말 표현이 일치하지 앞는 젓은?"
정확도: 95% (오타 2개: "않는" → "앞는", "것" → "젓")
```

### 다국어 처리
```
영어:   "Plants depend on sunlight and water"
OCR:    "Plants depend on sunlight and water On"
정확도: 98%

한국어: "식물들은 햇빛과 물에 의존한다"
OCR:    "식물들은 햇빛과 물에 의존한다"
정확도: 100%
```

### 도전적인 케이스
```
원본:   "There are a few dishes in the sink"
OCR:    "There Vfantasy different a few dishes in the sink are"
정확도: 70% (단어 순서 혼란)
```
