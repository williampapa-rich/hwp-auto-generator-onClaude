주요 특징:

1. 🎯 PyMuPDF (fitz) 기반 PDF 처리
   - 한국어 안정적 처리
   - 블록 단위 bbox 제공
   - 사각형/이미지 감지 기능

2. 🎨 지능형 레이아웃 분석
   - 동적 컬럼 경계 탐지
   - 15px dead-zone으로 경계 근처 오분류 방지
   - 2단 읽기 순서 (헤더/좌/우/푸터)

3. 📦 박스 텍스트 자동 분류
   - text_box: 흰 테두리 (지문)
   - condition_box: 회색/음영 (조건, 보기)
   - text: 박스 없음

4. 🤖 LLM 기반 의미 추출
   - Gemini API 활용
   - 재시도 로직 (3회)
   - Pydantic 검증

5. 🏗️ 모듈화된 아키텍처
   - 각 단계 독립적 테스트 가능
   - 설정 중앙화
   - 확장 가능한 구조

════════════════════════════════════════════════════════════════

빠른 시작:

1. 설치
   $ pip install -r requirements.txt

2. API 키 설정
   $ cp .env.example .env
   (GEMINI_API_KEY 입력)

3. 기본 검증 (LLM 없이)
   $ python test_parser.py sample.pdf

4. 완전한 파싱
   $ python exam_parser/main.py sample.pdf

결과:
- output/result.json
- images/*.png
